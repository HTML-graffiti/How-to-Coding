The Coverpage That Seems Like a cover of the book

開いた本のようなウェブページ

rel="stylesheet" href="style.css" /

#cover を フルスクリーンサイズで固定し、 #left と #right を float で左右にレイアウト
#left と #right の width と height を調整し、 transform で開いた本のように傾けます。

h1#title の writing-mode を縦書きにして、 #left と #right の間に回り込ませ、フォントサイズと余白を調整します。

id="cover">

開始タグに id="cover" を設定した body の中に、

id="left">
表紙になる main 要素 (#left)

id="right">
裏表紙になる article 要素 (#right)

id="title">
背表紙になる h1 要素 (#title)

を記述します。
include('呼び出したいページのURL');

PHP の include 式を使って、 #left と #right に 特定のファイルを呼びだします。

スクリーンサイズが 820px 以下の場合に適応するスタイルシート

rel="stylesheet" href="mobile.css" media="screen and (max-width: 820px)"/

#cover の position を static に、 #left と #right の幅を 100% に変更
通常フローに沿ってページの上から、#left・#right の順に要素をレイアウト・ h1#titleは、ページ右上の position に fixed します。 

    #cover {
      position: static;
      width: 100%;
      height: auto;
      max-height: auto;
    }

    #left,
    #right {
      clear: both;
      float: none;
      position: relative;
      padding: 5% 10% 5% 5%;
      margin-top: 0;
      width: 100%;
      height: auto;
      min-height: -webkit-fill-available;
      transform: rotate(0) skew(0, 0);
    }

    h1#title {
      font-size: 4vh;
      position: fixed;
      top: 2.5%;
      right: 2.5%;
      padding: 0;
      height: 100%;
    }

    @media screen and (max-width: 550px) {
      h1#title {
        font-size: 4.5vw;
      }
    }
