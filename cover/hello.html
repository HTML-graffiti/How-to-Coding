開いた本のようなウェブページ

Index.html 内に、 ID名#left・#rightと名付けた2つのDIV要素を記述し、JavaScript によって2つの異なるウェブページを呼び出し、 開いた本のように CSS でスタイリングするテンプレートを制作しました。

Index.html 

  <!DOCTYPE html>
   <html lang="ja">
   <head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title> Put in The Title of this page </title>
   <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
   <script type="text/javascript">
   $(function(){
       $("#left").load("hello.html");
       $("#right").load("book.php");
   })
   </script>

   <link rel="stylesheet" href="style.css"/>

   <style type="text/css">
   </style>
   </head>
   <body>
   <h1 id="title">Put in The Title of this page</h1>
   <div id="cover">
   <div id="left"></div>
   <div id="right"></div>
   </div>
   </body>
   </html>
