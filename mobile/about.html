コンピューターとモバイルで表示内容を変える

ウェブサイトを閲覧するウインドウサイズによって、表示内容を変える（レスポンシブと呼ばれる）CSS を紹介します。

はじめに、HEAD要素内にHTMLをレスポンシブ対応にする メタデータ を記述します。

HEAD要素内に、このコードを記述しなければ、レスポンシブ対応のウェブサイトは成立しません。

<meta name="viewport" content="width=device-width, initial-scale=1.0">

この一行を加えるだけで、ウェブサイトをレスポンシブ対応にできるとても便利な魔法のようなコードです。
HTML・PHP、ウェブサイトを構築するどんなドキュメントファイルにもこのコードを記述していて損はないはずです。

コンピューターとモバイルで表示内容を変えるCSS

id="canbe" (#canbe) は、コンピューターに表示する要素、id="mobile" (#mobile) は、モバイルに表示する要素、id="about" (#about) は、どちらのブラウザでも表示できるが、モバイルでは操作できない要素です。


#mobile {display:none;}

@media screen and (max-width: 500px){
#canbe {display:none;}
#mobile {display:block;}
#about {pointer-events:none;}
}

CSS #mobile {display:none;} は、すべてのブラウザに関係するCSSで、

@media screen and (max-width: 500px){

より下は、ウインドウサイズ 500px 以下のブラウザに対応するCSSです。

display:none;
要素の存在を消し、表示しないようにします。

pointer-events:none;
要素の選択・操作を出来ないようにします。

以上2つの指示によって、ウインドウサイズ 500px 以下のブラウザでは id="canbe" (#canbe) 要素は表示されず、ウインドウサイズ 500px 以上のブラウザでは表示されない id="mobile" (#mobile) 要素が表示され、id="canbe" (#canbe) 要素の選択・操作ができないようになります。

指定したID内の文字を自動で入れ替える JavaScript

Change the Words 

<h1 id="change"></h1>

<script>
var text = ["This","website","cannot", "be", "browse", "from", "Mobile" ];
var counter = 0;
var elem = document.getElementById("change");
var inst = setInterval(change, 1000);

elem.innerHTML = text[counter];

function change() {
  elem.innerHTML = text[counter];
  counter++;
  if (counter >= text.length) {
    counter = 0;
  }
};
</script>

このコードは、id="change"(#change) のH1要素内に、テキストを呼び出し、一定期間で呼び込んだ言葉を入れ替えるJavaScriptです。

BODY要素内に上記のコードを記述すると、"This","website","cannot", "be", "browse", "from", "Mobile" の7つの単語が自動的に入れ替わります。

※ 入れ替わる言葉を変える際は、"" の中の単語を書き換えてください。
"" と "" の間には , を記述してください。
（最後の "" の後に , をつけると、うまく動作しません。）

var inst = setInterval(change, 1000);
(change, 1000) 内の数字を小さくすると早く、大きくすると遅く、言葉が入れ替わります。
