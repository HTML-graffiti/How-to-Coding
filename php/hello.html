PHP: Hypertext Preprocessor

PHP マニュアル 日本 PHP ユーザ会 




初歩的な例：PHPを使ってHTMLにコードを埋め込む

<?php
echo "PHP: Hypertext Preprocessor";
?>

PHP を HTML に埋め込み、ページの内容を生成 したり、フォームからデータを取得 したり、動的にウェブサイトを構築することが出来ます。

※ PHP を使用する文書ファイルの拡張子は、.html ではなく .php を用います。


CSVに書き込んだ内容を、HTMLに呼び出す
/coding/php/csv/


フォームから内容を取得し、CSVに書き込む
/coding/submit/


自動返信機能付き投稿フォーム
/coding/submit/contact/
