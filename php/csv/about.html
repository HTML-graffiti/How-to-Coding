解説

＜?=h($row[0])?＞ 部分に、CSV に記述した内容を呼び出す PHP です。

※ カレンダーやリストページなど、表示内容を変更し、レイアウトを繰り返すことに適したコードです。

例えば、

＜span＞A＜/span＞ ＜span＞B＜/span＞ ＜span＞C＜/span＞

のように、HTML内に異なる内容の同じ要素を繰り返す場合、HTMLから編集したい箇所を探し、その部分を書き換えることが必要です。

このコードは、頻繁にコードを書き換えたい箇所に PHP を埋め込み、CSVファイルを書き換えるだけで指定の箇所を簡単に更新／編集するためのコードです。


<?php if (!empty($rows)): ?>
<?php foreach ($rows as $row): ?>
<span><?=h($row[0])?></span>
<?php endforeach; ?>
<?php else: ?>
<?php endif; ?>

BODY要素内の ＜span＞text＜/span＞を表示したい箇所に、上記のコードを記述します。

?phpで始まる複数の行は、CSVを呼び出すために必要なコードです。
この数行は編集しないで、このまま使用してください。深く理解しなくても問題ないです。

CSVを編集すると、

<span><?=h($row[0])?></span>

が繰り返され、＜?=h($row[0])?＞ 部分にCSVに記述したテキストが呼び出されます。
この仕様により、簡単に項目を増やしたり減らしたり、項目の内容を書き換えることが出来ます。

PHPファイルの冒頭に、CSVを呼び出すための PHPを記述します。


<?php
function h($str) {
    return htmlspecialchars($str, ENT_QUOTES, 'UTF-8');
}

$text = (string)filter_input(INPUT_POST, 'text');

$fp = fopen('log.csv', 'a+b');
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    flock($fp, LOCK_EX);
    fputcsv($fp, [$text,]);
    rewind($fp);
}
flock($fp, LOCK_SH);
while ($row = fgetcsv($fp)) {
    $rows[] = $row;
}
flock($fp, LOCK_UN);
fclose($fp);

?>

重要な部分は、「$fp = fopen('log.csv', 'a+b');」 と、「$text = (string)filter_input(INPUT_POST, 'text');」・「fputcsv($fp, [$text,]);」の3箇所です。

このコードのすべてを理解しようと思うと、難しいと感じてしまうと思います。
すべてを理解しなくても、この3箇所を理解するだけで、様々なウェブサイトを作成することが出来ます。

$fp = fopen('ファイル名.csv', 'a+b');
PHPに呼び出すCSVファイル名を指定するコードです。
（Sampleページでは、log.csvというファイルを呼び出しています。）

$項目名 = (string)filter_input(INPUT_POST, '項目名');
PHPを用いて呼び出したいCSVに記述する項目を指示します。
（Sampleページでは、textという項目名のテキストを呼び出しています。）

※ 重要 ※
項目名は、どんな英単語でも構いません。
$ の後 と '' 内 の項目名が同じでなければうまく動作しません。

fputcsv($fp, [$項目名,]);
fputcsv($fp, のあとの []内を例に、CSVに呼び出テキストを記入します。

※ 重要 ※
fputcsv($fp, のあとの [$項目名] と、
先に説明した、$項目名 = (string)filter_input(INPUT_POST, '項目名'); すべての項目名が同じ英単語でなければ、うまく動作しません。
