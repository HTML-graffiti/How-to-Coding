BODY要素 

<div id="hello">
<b>Hello こんにちは</b>
<p>This is The <u data-click="how">Page</u>
<b id="how" class="open">About
<u data-click="to">How to Coding</u>
<i id="to" class="open">
<a href="/" target="_blank" rel="noopener noreferrer">This Page</a>
</i>
</p>
</div>

id="hello" と名付けた DIV: コンテンツ分割要素 の中にある P: 段落要素 の中に、テキストを装飾する3つの要素 U: 非言語的注釈 (下線) 要素 B: 注目付け要素 I: 興味深いテキスト要素 を用意し、CSS と JavaScript で 表示／非表示を指定します。

＜u data-click="ID名"＞ この中に書いたテキストをクリックすると＜/u＞

CSSで非表示にしてた id="ID名" class="open" 要素 が JavaCcript によって表示されます。

※ 非表示要素を表示するトリガーとなる U要素の data-click="ID名" と 非表示要素の id="ID名" を揃え、HTML内にあるその他の要素に同じ ID名 を使わないように注意しましょう。

※ ここでは、要素の表示／非表示に関連するCSSのみを解説します。


#hello .open {display:none;}

要素の表示方法を指示する DISPLAYプロパティ を display:none; と指定し、ID名 #hello 内にある CLASS名 .open 要素を非表示にします。

CSSプロパティ DISPLAY については /html/divspan/ で解説します。


#hello u {cursor:help;}

非表示要素を表示するトリガーとなる ID名 #hello 要素内のU要素 に マウスを置くとカーソルが？に変化します。 CURSORプロパティ

open_close.js


$(function(){
    $("#hello u").on("click", function(){
    var id = $(this).data("click");
    $("#" + id).show(100);
    })
});
