テキスト要素の書体をスタイリングするCSSを解説します。

font-family
要素のフォントファミリ名・総称ファミリ名をスタイリングする

要素名 {font-family:"フォントファミリ名",総称フォント名;}

複数の要素に同じスタイリングを指定する／代替フォントを指定する場合
要素名,要素名 {font-family:"フォントファミリ名","フォントファミリ名",総称フォント名;}

連続して要素名／フォント名を記述する場合は、 , で要素名／フォント名を区切る。
（最後の要素名／フォント名のあとに , をつけるとCSSが動作しません。最後の要素名／フォント名のあとには , をつけないように注意しましょう。）


p,
h4 {
    font-family: Arial, sans-serif;
}

h1,
h5 {
    font-family: 'Times New Roman', Times, serif;
}

h2,
h6 {
    font-family: 'Comic Sans MS', cursive;
}

h3,
span {
    font-family: 'Courier New', Courier, monospace;
}

Tools

全標準フォント一覧

Mac・Windowsの標準フォントファミリ一覧できるウェブサイトです。
OSごとの表示の違いも確認できるので、とても便利です。

主な総称ファミリ名
serif
明朝体（終わりが止めや払い、または実際の活字のセリフの形をした角を持つ字形です。）
 sans-serif
ゴシック体（平たい角の端を持つ字形です。）
 cursive
筆記体（続け書き、もしくは、イタリック体の特性以上に、その他の筆記体の特性があります。）
 fantasy
fantasy フォントは、遊び心に溢れた文字表現を含む、主として装飾的なフォントです。
 monospace
すべての字が同じ幅を持つ字形です。


font-style
フォントの書体「通常体 (normal)、筆記体 (italic)、斜体 (oblique) 」を指定する


h1,
h5,
b,
s,
strong {
    font-style: normal;
}

h2,
i,
samp,
sub {
    font-style: italic;
}

h3,
h6,
q,
small,
sup {
    font-style: oblique;
}

h4 {
    font-style: oblique -90deg;
}

プロパティ値
normal
normal に分類されているフォントを選択します。
 italic
本質的に筆記体であり、スタイル付けされていない書体より水平方向の幅が小さいのが一般的です。
 oblique
たいてい、通常のフェイスを傾けただけです。
 oblique -90 ~ 90 deg
oblique として分類されているフォントを選択し、テキストの傾きの角度を指定します。

フォントの大きさ・太さ


h1 {
    font-size: 5vw;
    font-weight: 500;
}

h2 {
    font-size: 22px;
    font-weight: bolder;
}

h3 {
    font-size: 3rem;
    font-weight: 700;
}

h4 {
    font-size: 4em;
    font-weight: lighter;
}

プロパティ
font-size
フォントの大きさを定義します。
 font-weight
フォントの太さ (あるいは重み) を指定します。

変化形


h1 {
    font-stretch: condensed;
    font-variant: 500;
    line-height: normal;
}

h2 {
    font-stretch: 50%;
    font-variant: normal;
    line-height: 2.5;
}

h3 {
    font-stretch: expanded;
    font-variant: no-common-ligatures proportional-nums;
    line-height: 150%;
}

h4 {
    font-stretch: ultra-expanded;
    font-variant: common-ligatures tabular-nums;
    line-height: 2;
}

h5 {
    font-stretch: 100%;
    font-variant: small-caps slashed-zero;
    line-height: 100%;
}

h6 {
    font-stretch: 25%;
    font-variant: common-ligatures small-caps;
    line-height: 50%;
}

プロパティ
font-stretch
フォントファミリによっては追加のフェイスを提供しており、通常より狭い文字 (condensed フェイス)、通常より広い文字 (expanded フェイス) などを指定できます。
 font-variant
CSS の一括指定プロパティで、フォントのすべての変化形を設定することができます。
