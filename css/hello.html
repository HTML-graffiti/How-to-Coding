CSS セレクター 擬似クラス Pseudo-classes 詳細度 Specificity

HTML で構築した内容の表示方法を指定するコードが CSS です。
例えばウェブサイトを家だと考えた場合、必要な部屋の数・机・棚・椅子などをHTMLで記述し、それらの色や形、どこに置くかなどをCSSで指定するというイメージです。

CSSは、HEAD要素内 ＜style＞ここにCSSを記述します＜/style＞ 、または、制作したCSSファイルを HTMLにLINK HEAD要素内に＜link href="CSSファイル名" rel="stylesheet"＞を記述 します。

CSSの記述方法


  p {
    color: green;
    font-size: 72px;
    padding: 10px;
    margin-right: 20px;
  }

これは、「P要素」の 色を「緑」に、 文字サイズを「72px」に、 要素内の余白を「10px」に、 要素外の右に余白を「20px」 に指定するCSSコードです。

要素名「セレクター (Selector)」、要素の規則「宣言 (Declaration)」を{　}内に記述します。
「宣言 (Declaration)」は、要素をスタイル付けするための方法「プロパティ (Property)」:「プロパティ値 (Property value)」;を記述します。

  • 規則セットは中括弧 {　} で囲む

  • 宣言内では、コロン : を使用してプロパティと値を分離する

  • 規則セット内でセミコロン ; を使用して、それぞれの宣言と次の規則を区切る

    • 以上3つのルールを理解できれば、あとはいろいろなCSSプロパティを知り、コーディングを楽しむだけです。

要素名で規則を宣言する他に、HTML内の要素に共通の属性名をつける CLASS と ID を用いて規則を宣言することもできます。


例えば、


    <div class="special">
      <!-- HTML here -->
    </div>

    <div id="about">
      <!-- HTML here -->
    </div>

    <p class="special">クラス名スペシャルのP要素</p>

    <p id="about">ID名アバウトのP要素</p>

のように、異なる要素に共通の属性名をつけ、


  .special {
    /* CSS here */
  }

  #about {
    /* CSS here */
  }

と規則を宣言し、要素ではなく属性名で共通のスタイルを付けることもできます。

CSSでは、CLASS名を「.CLASS名」 と記述し、ID名を「#ID名」と記述します。
