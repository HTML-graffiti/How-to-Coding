CSS は、ウェブページのスタイルを指定するための言語です。
例えばウェブサイトを家だと考えた場合、必要な部屋の数・机・棚・椅子などを HTML に記述し、それらの色や形、どこに何を置くかなどをCSSでスタイリングします。

CSSは、HTML の HEAD要素内 ＜style＞ここにCSSを記述＜/style＞ または、CSSファイルを HTMLにLINK HEAD要素内にリンクするCSSファイル＜link href="CSSファイル名.css" rel="stylesheet"＞を記述 し、HTMLと組み合わせて使用します。

CSSの記述方法


  p {
    color: green;
    font-size: 72px;
    padding: 10px;
    margin-right: 20px;
  }

これは、「P要素」の 色を「緑」に、 文字サイズを「72px」に、 要素内の余白を「10px」に、 要素外の右に余白を「20px」 に指定するCSSです。

どの部分に対してのCSSなのかを、要素名「セレクター (Selector)」で表し、どんなスタイリングを行うかを、要素の規則「宣言 (Declaration)」で指示します。
「宣言 (Declaration)」には、要素をスタイル付けするための方法「プロパティ (Property)」:「プロパティ値 (Property value)」;を記述します。

  • 規則セットは中括弧 {　} で囲む

  • 宣言内では、コロン : を使用してプロパティと値を分離する

  • 規則セット内でセミコロン ; を使用して、それぞれの宣言と次の規則を区切る

    • 以上3つのルールを理解できれば、CSSの基本は完璧です。
    下記の目次から様々なCSSプロパティとHTMLの関係を学んでいきましょう。

要素名で規則を宣言する他に、要素に特定の名前をつけるグローバル属性と呼ばれるHTML言語 CLASS と ID を用いて規則を宣言することもできます。


例えば、


    <div class="special">
      <!-- HTML here -->
    </div>

    <div id="about">
      <!-- HTML here -->
    </div>

    <p class="special">クラス名スペシャルのP要素</p>

    <p id="about">ID名アバウトのP要素</p>

のように、異なる要素に共通の属性名をつけ、


  .special {
    /* CSS here */
  }

  #about {
    /* CSS here */
  }

要素名ではなく属性名で共通のスタイルを付けることもできます。

※ 詳しくは、A要素とCLASS・ID属性 ページで解説します。
