A要素とCLASS・ID属性 ページで解説した、A要素のテキストを線や点などで装飾する CSS を解説します。

初めに、このページの左に埋め込まれた HTML 内のすべてのA要素を A要素とCLASS・ID属性 の解説に沿ってフルディスプレイサイズで表示。
ディスプレイ値 を display:inline-block または、 display:block に指定し、要素の大きさ・余白 をスタイリングします。


#click {
    display: block;
}

#click a {
    display: inline-block;
}

CSSプロパティ「ディスプレイ」については、DIVとSPAN要素 で詳しく解説します。


#click a {
    width: 90%;
    height: 90vh;
    padding: 2.5vh 2.5%;
    margin: 2.5vh auto;
}

プロパティ値
width
要素の幅を設定するプロパティ
 height
要素の高さを指定するプロパティ
 margin
要素の周りに追加の領域を作成するプロパティ
 padding
要素の内部に追加の領域を作成するプロパティ

width min-width max-width height min-height max-height margin padding

要素の大きさ・余白について、コンピューターとモバイルで表示内容を変える で詳しく解説します。




.none {
    text-decoration: #000 underline solid;
    text-decoration-skip-ink: auto;
    text-emphasis: filled red;
    -webkit-text-emphasis: filled red;
}

.double {
    text-decoration: blue underline overline 2px double;
    text-decoration-skip-ink: none;
}

.dotted {
    text-decoration: #000 underline 5px dashed;
    text-decoration-skip-ink: none;
    text-emphasis: 'x' red;
    -webkit-text-emphasis: 'x' red;
}

text-decoration
テキストの装飾的な線の表示を設定するプロパティ
 text-decoration-skip-ink
上線や下線が文字のアセンダーやディセンダーを通過するときにどのように引くのかを指定するプロパティ
 text-emphasis
(空白や制御文字を除く) テキストに圏点を適用するプロパティ


字間 と 大文字・小文字 をスタイリングするプロパティ


.shadow {
    text-indent: 2vw;
    letter-spacing: 0.5vw;
}

.none {
    white-space: pre-wrap;
    word-spacing: 120%;
    word-break: break-all;
}

text-indent
テキストの行の前に置く空白 (字下げ) の幅を設定するプロパティ
 letter-spacing
テキストの水平方向の字間のスペースに関する挙動を設定するプロパティ
 white-space
要素内のホワイトスペースをどのように扱うかを設定するプロパティ
 word-spacing
タグや単語の間隔に関する挙動を指定するプロパティ
 word-break
改行しなければテキストがコンテンツボックスからあふれる場合に、ブラウザーが改行を挿入するかどうかを指定するプロパティ
 



.solid {
    text-transform: capitalize;
    text-shadow: -0.75vw -0.5vw 0 red;
}

.dotted {
    text-transform: lowercase;
    text-shadow: 0.25vw 0.25vw blue, -0.5vw -0.25vw #000, 0 0 2vw red;
}

text-transform
要素のテキストを大文字表記する方法を指定するプロパティ
 text-shadow
テキストに影を追加するプロパティ
