要素を装飾するために特定の名前をつけるグローバル属性と呼ばれるHTML言語を並べた HTML の表示位置・余白・囲い線などをスタイリングする CSS を解説します。

このページの左に埋め込まれた HTML 内のすべての要素の ディスプレイ値 を display:inline-block または、display:block に指定し、要素の 大きさ・余白 をスタイリングします。

例

#click, .demo {display:block;}
#click a {display:inline-block;}

CSSプロパティ「ディスプレイ」については、DIVとSPAN要素 で詳しく解説します。

例

#click, .demo {
padding:10%;
margin:auto;
width:75%;
}
.demo {
height:5rem;
}
#click a {
padding:1% 2.5%;
margin:2.5%;
}

プロパティ値
width
要素の幅を設定します。
 height
要素の高さを指定します。
 margin
要素の周りに追加の領域を作成します。
 padding
要素の内部に追加の領域を作成します。

width min-width max-width height min-height max-height margin padding

要素内のテキストの 表示位置 をスタイリングします。

例

.demo {
text-align:center;
vertical-align: middle;
}

text-align
ブロック要素または表セルボックスの水平方向の配置を設定します。
 vertical-align
インラインボックス、インラインブロック、表セルボックスの垂直方向の配置を設定します。
 text-indent
ブロック内のテキストの行の前に置く空白 (字下げ) の幅を設定します。

テキストデコレーション をスタイリングします。

例

.demo {
text-decoration:#000 underline dotted 5px;
text-decoration-skip-ink: none;
text-emphasis: filled red;
-webkit-text-emphasis: filled red;
}
.border {
text-decoration:blue underline overline 10px;
text-decoration-skip-ink: auto;
}
.radius {
text-decoration:red wavy underline 2px;
text-decoration-skip-ink: none;
text-emphasis: 'x';
-webkit-text-emphasis: 'x';
}
.none {
text-decoration:none;
}

text-decoration
テキストの装飾的な線の表示を設定します。
 text-decoration-skip-ink
上線や下線が文字のアセンダーやディセンダーを通過するときにどのように引くのかを指定します。
 text-emphasis
(空白や制御文字を除く) テキストに圏点を適用します。

字間 と 大文字・小文字 をスタイリングします。

例

.demo {
letter-spacing: 100%;
word-spacing: 100%;
}
.border {
letter-spacing: 150%;
word-spacing: 50%;
white-space: pre-wrap;
}
.radius {
letter-spacing: 75%;
word-spacing: 125%;
white-space: break-spaces;
}

letter-spacing
テキストの水平方向の字間のスペースに関する挙動を設定します。
 white-space
要素内のホワイトスペースをどのように扱うかを設定します。
 word-spacing
タグや単語の間隔に関する挙動を指定します。
 word-break
改行しなければテキストがコンテンツボックスからあふれる場合に、ブラウザーが改行を挿入するかどうかを指定します。

例

.demo {
text-transform: capitalize;
}
.border {
text-transform: uppercase;
}
.radius {
text-transform: lowercase;
}
.none {
text-transform: none;
}

text-transform
要素のテキストを大文字表記する方法を指定します。
