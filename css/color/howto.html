ページを更新するごとにランダムで色が変化するJavaScript

JavaScriptについてはまだ勉強中で、説明するほど理解できていません。


        function getRumRgba(){
          var clr = 'rgba(';
          for(i=0; i < 3; i++){
            clr = clr + Math.floor( Math.random()*255) + ',';
          }
          clr = clr + Math.floor( Math.random()*10)/10 + ')';
          return clr;
        }

色コードをRGBで指定し、1から255までの数字をランダムで生成し、


          $(function(){
            jQuery('.クラス名').css({'background':getRumRgba()});
          });
          
          $(function(){
            jQuery('.クラス名').css({'color':getRumRgba()});
          })

クラス名.randombgの背景色／クラス名.randomcolorの文字色を指定しています。
