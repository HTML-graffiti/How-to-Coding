「.both」「.pc」「.mobile」とCLASS名をつけたDIVとSPAN要素を羅列したHTMLの文字色・背景色をCSSでスタイリングする

はじめに、DIVとSPAN要素を区別するため、要素の境界 border を指定します。


div {
    border: thick double;
}

span {
    border: 2px dashed;
}

要素に境界線をスタイリングするBODERプロパティについては、 A要素とCLASS・ID属性 ページで詳しく解説します。



CLASS名「.both」「.pc」「.mobile」それぞれの要素に異なる 文字色・背景色 をスタイリングします。



.pc {
    color: #000;
    background-color: #fff;
    mix-blend-mode: hard-light;
}

.mobile {
    color: red;
    background-color: hsla(0, 0%, 0%, .75);
    mix-blend-mode: difference;
}

プロパティ値
color
要素のテキストやテキスト装飾における前景色の色の値を設定します。
background
色、画像、原点と寸法、反復方法など、背景に関するすべてのスタイルプロパティを一括で設定します。
background-color
要素の背景色を設定します。
mix-blend-mode
要素の内容物と親要素の内容物や要素の背景をどのように混合するかを設定します。

Tools

Color Names — HTML Color Codes 色見本 ～ カラーコード Picular Pigment Colorable CSS Gradient


SPAN要素の上をカーソル (マウスポインター) がホバー (通過)した時 :hover に、要素の色が反転するようCSSのfilterプロパティ値 invert()


span:hover {
    filter: invert(1);
}

.pc {
    filter: invert(0);
}

.mobile {
    filter: invert(75%);
}

色や画像にグラフィカル効果をかけるプロパティについては、 filter-function ページで詳しく解説します。

JavaScriptについてはまだ勉強中で、説明するほど理解できていません。


function getRumRgba(){
var clr = 'rgba(';
for(i=0; i < 3; i++){
clr = clr + Math.floor( Math.random()*255) + ',';
}
clr = clr + Math.floor( Math.random()*10)/10 + ')';
return clr;
}

色コードをRGBで指定し、1から255までの数字をランダムで生成し、


$(function(){
jQuery('要素').css({'background':getRumRgba()});
});

$(function(){
jQuery('要素').css({'color':getRumRgba()});
})
