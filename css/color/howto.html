「.both」「.pc」「.mobile」とCLASS名をつけたDIVとSPAN要素を羅列したHTMLの文字色・背景色をスタイリングするCSSを解説します。

はじめに、DIVとSPAN要素を区別するため、要素の境界 border をスタイリングします。


div {
border: thick double;
border-radius: 25% 10%;
}
span {
border: 2px dashed;
border-radius: 50% 20% / 10% 40%;
}

プロパティ値
border
要素の境界を設定します。
border-radius
要素の境界の外側の角を丸めます。

CLASS名「.both」「.pc」「.mobile」の要素に異なる 文字色・背景色 をスタイリングします。

例

div {
color:#000;
background: radial-gradient(#e66465, #9198e5);
}
span {
color:#000;
background: radial-gradient(#9198e5, #e66465);
}

.pc {
mix-blend-mode: color-dodge;
}
.mobile {
mix-blend-mode: luminosity;
}

プロパティ値
color
要素のテキストやテキスト装飾における前景色の色の値を設定します。
background-color
要素の背景色を設定します。
background
色、画像、原点と寸法、反復方法など、背景に関するすべてのスタイルプロパティを一括で設定します。
mix-blend-mode
要素の内容物と親要素の内容物や要素の背景をどのように混合するかを設定します。
radial-gradient()
2つ以上の色で構成されるグラデーションを作成します。

SPAN要素の上をカーソル (マウスポインター) がホバー (通過)した時 :hover に、要素の色が反転するようCSSのfilterプロパティ値 invert()


span:hover {
filter: invert(1);
}
.pc {
filter: invert(0);
}
.mobile {
filter: invert(75%);
}
