Sample

このテンプレートは、index.html・style.cssの2つのファイルで構成されます。
※ ウェブページに表示する画像は、それぞれにご用意ください。

独自ページを制作するために編集が必要な箇所は、HEAD・CLASS="COVER"・ID="INFOMATION" の3箇所です。

HEAD要素内の以下の部分を編集し、ページタイトル・ページに表示する画像、フォントなどを編集します。

HEAD
<meta name="description" content="ページの説明">
<title>ページタイトル</title>
<style type="text/css">

.cover {
font-family: "フォントファミリ名";
}
.cover h1 {
text-shadow:0 0 0.5vw #fff;
color:#000;
font-family: "フォントファミリ名";
}

#infomation h1 {
text-shadow:0 0 0.5vw #000;
color:#fff;
font-family: "フォントファミリ名";
}
#infomation span {
color:#000;
text-shadow:0 0 0.5vw #fff;
box-shadow:0 0 0.5vw #fff;
background-color:rgba(255,255,255,0.25);
}

.one {background-image:url(画像URL);}
.two {background-image:url(画像URL);}
.three {background-image:url(画像URL);}
.four {background-image:url(画像URL);}
.five {background-image:url(画像URL);}
.six {background-image:url(画像URL);}
.seven {background-image:url(画像URL);}
.eight {background-image:url(画像URL);}
</style>

ページ最上部に表示する情報を記述します。

CLASS="COVER"
<div class="cover one">
<span class="left">左上</span>
<span class="right">右上</span>
<h1><u>下線サブタイトル</u><br/>
タイトル
</h1>
</div>

ページ最下部に表示する情報を記述します。

ID="INFOMATION"
<div id="infomation" class="wide">
<h1>#INFOMATION H1<br/>
<a href="">A</a><br/>
<span>SPAN</span></h1>
<iframe src=""></iframe>
</div>

INDEX.HTML に STYLE.CSS をリンクし、ページをスタイリングします。
※ STYLE.CSS の編集は必要ありません。INDEX.HTMLと同じリポジトリ（フォルダ）に STYLE.CSS STYLE.CSS を公開しご使用ください。

CSSプロパティ background-image:url(); を使って、要素の背景に横長・縦長・正方形の画像を埋め込み、ダウンロードされづらいフォトアルバムページを作る
