主なフォーム部品

フォームを構成する BODY要素 を解説します。

FORM要素 内に、フォームを構築する部品のコードを記述します。

<form action="complete.php" method="post" id="form"> </form>

formの開始タグ内に記述している action="complete.php" は、フォームのデータを送信後、移動するリンク先を指定するコードです。 method="post" については、ちゃんと理解していません。

コードを見る


<form action="complete.php" method="post" id="form">
<input type="date" name="date" required>
<input type="text" name="title" placeholder="タイトル" required>
<textarea name="text" placeholder="説明文" required></textarea>
<input type="password" placeholder="パスワード" name="password">
<p class="radio">Link（1つだけ選択が可能）
<input type="radio" name="link" value="block" required>あり
<input type="radio" name="link" value="none" checked="checked" required>なし
</p>
<p class="radio">（複数選択が可能）
<input type="checkbox" name="checkbox" value="a" required>a
<input type="checkbox" name="checkbox" value="b" required>b
<input type="checkbox" name="checkbox" value="c" required>c
</p>
<input type="text" name="url" placeholder="リンク先のURL">

<select name="tag">
<option value="">（1つだけ選択が可能）</option>
<option value="a">A</option>
<option value="b">B</option>
<option value="c">C</option>
</select>
<button type="submit">投稿 POST</button>
</form>

主なフォーム部品
日付選択フォーム（input type="date"）・テキストフォーム（input type="text"）・パスワードフォーム（input type="password"）・ラジオボタン（input type="radio"）・チェックボックス（input type="checkbox"）
複数行の入力欄（textarea）
プルダウンメニュー（select | option）
送信ボタン（button type="submit"）

詳しくは、 HTML5/フォーム/form要素 入力フォームを作る input要素 フォームを構成する様々な入力部品 を参照ください。

フォーム部品共通のルール
開始タグに required と記述したフォームは、入力必須
placeholder="ここに入力したテキストが、例として空欄フォームに表示される"
value="ここに入力したテキストは、初期入力値となる。（valueで初期入力値を指定した場合、テキストフォーム／入力欄を未記入のまま送信すると、設定した初期入力値が記録されます。）
※ ラジオボタン・チェックボックス・プルダウンメニューなどのフォームで選択した結果は、valueで設定した初期入力値が記録されます。"


フォームを装飾するCSS

使用したCSSプロパティは padding margin width height border border-radius font-size です。

特にこだわったスタイリングではなく、全てのフォーム部品をウインドウサイズに合わせているだけのシンプルなCSSです。

コードを見る


#form {
  overflow-x:hidden;
  padding:2.5% 0;
}
#form input[type="date"],
#form input[type="text"],
#form input[type="password"],
#form input[type="url"],
#form input[type="reset"],
#form textarea,
#form button,
#form select,
#form .radio {
  font-size:1rem;
  border:1px solid;
  border-radius:0.25rem;
  padding:2%;
  margin:1.25% 2.5%;
  width:90%;
}
#form textarea {height:10rem;}
#form button,
#form select
{width:95%;}
