マルチメディアとその埋め込み方

MDN Web Docs を参考に、ウェブページにマルチメディアを埋め込みます。

画像を埋め込む HTML をスタイリングする CSS

画像埋め込み要素 IMG の src 属性に埋め込みたい画像へのパスを入れます。
alt 属性は画像のテキストによる説明で、必須ではありませんがアクセシビリティのために非常に有用です。 


<img class="grayscale" src="http://praytimes.pe.hu/img/inn/000.JPG" alt="プレイタイムズ">
<img class="sepia" src="http://praytimes.pe.hu/img/inn/004.JPG" alt="キラキラゴッド">
<img class="contrast" src="http://praytimes.pe.hu/img/inn/006.JPG" alt="スーツケースとズボン">

加えて、画像に異なる視覚効果を加えるため、CLASS名をつけています。

3つの画像を横並びに表示するように画像の幅を30%に指定
position・z-index・transformプロパティでホバー時に画像サイズ／重ね順が変化するCSSを記述します。


  img {
    width:30%;
    position:relative;
}
  img:hover {
    z-index:10;
    transform: scale(2.5);
}
  
  .grayscale {
  filter: grayscale(100%);
  transition:.5s all;
  }
  .sepia {
  filter: sepia(100%);
  transition:.5s all;
  }
  .contrast {
  filter: contrast(500%);
  transition:.5s all;
  }
  .grayscale:hover {
  filter: grayscale(0%);
  transition:.5s all;
  }
  .sepia:hover {
  filter: sepia(0%);
  transition:.5s all;
  }
  .contrast:hover {
  filter: contrast(100%);
  transition:.5s all;
  }

加えて、filter・transitionプロパティで視覚効果アニメーションをスタイリングします。

動画と音声を埋め込む HTML をスタイリングする CSS

動画埋め込み要素 VIDEO・埋め込み音声要素 AUDIO の src 属性の中に表示したいメディアへのパスを含めます。
開始タグ内に controls と記述するとコントロールインターフェイスが表示されます。
その他にも、自動再生 autoplay、繰り返し loop、ミュート muted、全画面表示禁止 playsinlineなどの属性をつけることができます。


<video src="http://vg.pe.hu/jp/dm/pr/sample.mov" loop autoplay muted></video>

<audio controls>
  <source src="http://la.chottocrazy.pe.hu/collection/female%201.mp3" type="audio/mp3">
  Sorry, your browser doesn't support embedded Audios.
</audio>

動画と音声の幅を40%に指定し、中央に揃うよう要素の外側に空白を追加します。


  video,
  audio {
  width:40%;
  margin:2.5% 30%;
  }
  
  video {
  transform: matrix(1, 1.1, -1, 1, 10, 10);
  }
  
  audio {
  transform: skew(30deg, 20deg);
  }

transform-functionプロパティで要素の外見の変形をスタイリングします。

HTML ページに他のページ／PDFを埋め込む HTML をスタイリングする CSS

HTMLの書き方は、画像・動画・音声コンテンツの貼り付け方法とほぼ同じです。


<iframe src="http://newlifecollection.pe.hu/sign/">
  Sorry, your browser doesn't support embedded iframe.
</iframe>

<object data="http://goout.pe.hu/relax/essay.pdf" type="application/pdf" typemustmatch>
  Sorry, your browser doesn't support embedded odject.
</object>

要素の幅と高さを指定し、インフレーム要素を楕円にスタイリングします。


  iframe {
  width:80%;
  height:30vw;
  margin:10%;
  border:none;
  border-radius: 10% 30% 50% 70%;
  }
  
  object {
  width:100%;
  height:100vh;
  }
