マルチメディアとその埋め込み方

MDN Web Docs を参考に、ウェブページにマルチメディアを埋め込みます。

画像を埋め込む HTML をスタイリングする CSS

画像埋め込み要素 IMG の src 属性に埋め込みたい画像へのパスを入れます。
alt 属性は画像のテキストによる説明で、必須ではありませんがアクセシビリティのために非常に有用です。 


    <img class="grayscale" src="画像URL" alt="代替テキスト">
    <img class="sepia" src="画像URL" alt="代替テキスト">
    <img class="contrast" src="画像URL" alt="代替テキスト">

加えて、画像に異なる視覚効果を加えるため、CLASS名をつけています。

3つの画像を横並びに表示するように画像の幅を30%に指定
加えて、ホバー時に視覚効果が変化するCSSを記述します。 


img {
    width: 30%;
}

.grayscale {
  filter: brightness(2) grayscale(100%) invert(100%);
}

.sepia {
  filter: sepia(100%) drop-shadow(16px 16px 10px black) opacity(90%)
}

.contrast {
  filter: contrast(500%) hue-rotate(90deg) saturate(500%);
}

.grayscale:hover {
  filter: brightness(1);
  filter: grayscale(0%);
  filter: invert(0%);
}

.sepia:hover {
  filter: sepia(0%);
  filter: drop-shadow(0 0 0 black);
  filter: opacity(100%)
}

.contrast:hover {
  filter: contrast(100%);
  filter: hue-rotate(0deg);
  filter: saturate(100%);
}

Filter Function

CSSプロパティで、ぼかしや色変化などのグラフィック効果を要素に適用
brightness
明るさを明るくしたり暗くしたりします。
 contrast
コントラストを調整します。
 drop-shadow
要素に影をスタイリングします。
 grayscale
グレースケールに変換します。
 hue-rotate
入力画像の色相を回転させます。
 invert
サンプルを反転します。
 opacity
サンプルに透過度を適用します。
 saturate
彩度を変化させます。
 sepia
入力画像をセピア調にします。

動画と音声を埋め込む HTML

動画埋め込み要素 VIDEO・埋め込み音声要素 AUDIO の src 属性の中に表示したいメディアへのパスを含めます。
開始タグ内に controls と記述するとコントロールインターフェイスが表示されます。
その他にも、自動再生 autoplay、繰り返し loop、ミュート muted、全画面表示禁止 playsinlineなどの属性をつけることができます。


    <video src="" loop autoplay muted></video>

    <audio controls>
      <source src="" type="audio/mp3">
      Sorry, your browser doesn't support embedded Audios.
    </audio>

    <iframe src="">
      Sorry, your browser doesn't support embedded iframe.
    </iframe>
    
    <object data="" type="application/pdf" typemustmatch>

    </object>
