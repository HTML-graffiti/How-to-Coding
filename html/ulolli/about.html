UL と OL は、どちらもともに、箇条書きのリスト を制作する要素です。

UL または OL 内の LI要素 が、リスト内の項目で、UL 内の LI には順序がなく、LI の前には「・」が表示され、OL 内の LI の前には 上から順番に数字が振られます。

Library LI: リストの項目

UL と LI コードを見る

<ul>
<li></li>
<li></li>
<li></li>
</ul>

Library UL: 順序なしリスト要素

OL と LI コードを見る

<ol>
<li></li>
<li></li>
<li></li>
</ol>

Library OL: 順序付きリスト要素
