style.css

CLASS名 circle の UL要素 内の 項目を表す LI要素 のレイアウト方法を CSS フレックスボックスレイアウト CSSフレックスボックス(display:flex) にするCSSです。 


    .circle {
      list-style-type:hiragana-iroha;
      display:-webkit-flex;
      display:flex;
      justify-content: space-evenly;
      -webkit-flex-wrap:wrap;
      flex-wrap:wrap;
      -webkit-align-items:center;
      align-items:center;
    }
    .circle li {
      font-size:5.5vw;
      text-align:center;
      margin:1rem;
      width:55%;
      position:relative;
      transition:.5s all;
    }
    .circle li span {
      display:inline-block;
      width:75%;
      height:75%;
      padding:0.25rem;
      border:solid 1px;
    }
    .circle li b {
      font-weight:200;
      display:inline-block;
    }
    .circle li:hover {
      font-size:7.5vw;
      width:45%;
    }

Library justify-content flex-wrap align-items

radius.css

CLASS名 radius の 要素の囲い線を楕円にし、 animation を加えるCSSです。


    .radius {
      border-radius: 50% 40% 35%;
      animation: radius 10s linear infinite;
      transform: scale(1);
    }

    @keyframes radius {
      0% {
        border-radius: 50% 40% 35%;
        transform: scale(1);
      }
      25% {
        border-radius: 40% 35% 40%;
        transform: scale(1.02);
      }
      50% {
        border-radius: 35% 40% 50%;
        transform: scale(1);
      }
      75% {
        border-radius: 40% 50% 40%;
        transform: scale(0.98);
      }
      100% {
        border-radius: 50% 40% 35%;
        transform: scale(1);
      }
    }

Library border-radius transform

rotation.css

CLASS名 rotation の 要素を回転させる、 animation を加えるCSSです。


    .rotation {
      animation:5s linear infinite rotation;
    }
    .rotationX {
      animation:5s linear infinite rotationX;
    }
    .rotationY {
      animation:5s linear infinite rotationY;
    }

    @keyframes rotation{
      0%{ transform:rotate(0);}
      100%{ transform:rotate(360deg); }
    }
    @keyframes rotationX{
      0%{ transform:rotateX(0);}
      100%{ transform:rotateX(360deg); }
    }
    @keyframes rotationY{
      0%{ transform:rotateY(0);}
      100%{ transform:rotateY(360deg); }
    }

Library rotate rotateX rotateY
