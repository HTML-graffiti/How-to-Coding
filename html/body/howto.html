本文要素

このページでは、HTML の内 ブラウザに表示されるBODY要素の中に記述する代表的な要素を解説します。

もっと詳しく

BODY: 文書の本文要素


  <header id="header">
    <nav id="nav">
      <ol>
        <li><a>BODY</a></li>
        <li><a href="#header">HEADER</a></li>
        <li><a href="#nav">NAV</a></li>
        <li><a href="#main">MAIN</a></li>
        <li><a href="#footer">FOOTER</a></li>
        <li><a href="#address">ADDRESS</a></li>
      </ol>
    </nav>
  </header>
  
  <main id="main">
    <p>MAIN 要素</p>
    <a href="https://momdo.github.io/html/grouping-content.html#the-main-element" target="_parent"></a>
  </main>
  
  <footer id="footer">
  
    <address id="address">
      <p>
        <a href="mailto:we.are.pe.hu@gmail.com">email</a>
      </p>
    </address>
  
  </footer>

HEADER は、ページの見出し や NAV: ナビゲーションセクション要素 など、導入的なコンテンツを記述する、一般的にBODY要素の冒頭にある要素です。

NAV要素をスタイルングするCSS

  nav {
    position: fixed;
    top: 0;
    z-index: 10;
    padding: 5vh 0 2.5vh;
  }
  nav ol {
    list-style-type: none;
    padding-left: 0;
  }
  nav li {
    display: inline-block;
  }
  nav a::after {
    display: inline-block;
    color: #000;
    content: '>';
    font-size: 80%;
    font-weight: bold;
    padding: 0 3px;
  }

メニュー、目次、索引などを記述するリスト要素については Ul Ol Li ページを参照ください。

MAIN は、BODY要素の主要な内容を表す要素です。
HTML内には hidden 属性が指定されていない MAIN要素を2つ以上置くことはできません。

※ 要素名の後に hidden と記述すると、要素を非表示にすることができます。

MAIN要素をスタイルングするCSS

  main {
    padding: relative;
    height: 100vh;
    color: #fff;
    display: grid;
    place-items: center;
  }
  main a {
    display: block;
    position: absolute;
    width: 100%;
    height: 100%;
    background-color: rgba(0,0,0,0.25);
  }
  main a:hover {
    filter: blur(2rem);
  }

要素内の中央にテキストをスタイリングするCSSについては Grid Layout ページを参照ください。

FOOTER は、一般的にBODY要素の最後に表示する、著作者に関する情報や ADDRESS: 連絡先要素 などを記述する要素です。
