本文要素

このページでは、HTML の内 ブラウザに表示されるBODY要素の中に記述する代表的な要素を解説します。

もっと詳しく

BODY: 文書の本文要素


    <header id="header">

        <nav id="nav">
            <ol>
                <li><a>HEAD 要素</a></li>
                <li><a href="#main">MAIN 要素</a></li>
                <li><a href="#footer">FOOTER 要素</a></li>
            </ol>
        </nav>

    </header>

    <main id="main">
        MAIN 要素
    </main>

    <footer id="footer">

        <address id="address">
          <p><a href="hello.html">FOOTER 要素</a></p>
        </address>

    </footer>

一般的には、HEAD要素にメニュー、MAIN要素に主要な内容、FOOTER要素にコピーライトなどのクレジットを記述します。

HEADER は、一般的にBODY要素の冒頭にある要素です。
ページの見出し や NAV: ナビゲーションセクション要素 など、導入的なコンテンツを記述します。

NAV要素をスタイルングするCSS

nav ol {
    list-style-type: none;
    padding-left: 0;
}

nav li {
    display: inline-block;
}

nav a::after {
    display: inline-block;
    color: #000;
    content: '>';
    font-size: 80%;
    font-weight: bold;
    padding: 0 3px;
}

メニュー、目次、索引などを記述するリスト要素については Ul Ol Li ページを参照ください。

MAIN は、HTML5/HTML Living Standardから導入された新要素。BODY要素の主要な内容を表します。
HTML内には hidden 属性が指定されていない MAIN要素を2つ以上置くことはできません。

※ 要素名の後に hidden と記述すると、要素を非表示にすることができます。

MAIN要素をスタイルングするCSS

main {
    height: 100vh;
    color: #000;
    display: grid;
    background-color: #ccc;
    place-items: center;
}

要素内の中央にテキストをスタイリングするCSSについては Grid Layout ページを参照ください。

FOOTER は、一般的にBODY要素の最後に表示する、著作者に関する情報や ADDRESS: 連絡先要素 などを記述する要素です。

FOOTER要素をスタイルングするCSS


footer {
    text-align: center;
}

footer a {
    display: block;
    padding: 5vh 2.5%;
    background-color: #000;
}

address {
    font-size: 2rem;
    font-family: fantasy;
}

address a[href^="mailto"]::before {
    content: "📧 ";
}

要素全体にリンクを広げるCSSについては A要素とCLASS・ID属性 ページを参照ください。
