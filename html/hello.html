Guide HTML の基本 HTML: HyperText Markup Language HTMLファイルを作成する

全ての HTML HyperText Markup Language は DOCTYPE ＜!DOCTYPE html＞ から始まります。
※ これは、この文書ファイルが HTML だということを宣言するコードです。

次に、HTMLの始まりを表す ＜html＞ HTML要素 ※ HTML内で使用されている言語「LANG属性」を日本語に指定 ＜html lang="ja"＞ することをおすすめします。（langはlanguageの略。jaはjapaneseの略。） から 終わりを表す ＜/html＞ の中に、2つの要素 HEAD と BODY を記述します。



<!DOCTYPE html>
<html lang="ja">
<head>
</head>
<body>
</body>
</html>

以上が、HTMLの基本です。

ページの基本要素 ウェブページは、HTMLの中にある HEAD と BODY に、記述されたさまざまな要素で構築されます。

要素は、
HTML 要素リファレンス ＜要素名＞ ここに表示内容を記述する＜/要素名＞ 要素名の前に/を入れ、その要素の終わりを表します。
または、＜要素名 属性=" 属性一覧 共通属性 "＞
どちらかのルールに基づき記述します。

BODY要素 は、ブラウザに表示されるウェブページの中身です。
Body内で使用する要素
例えば、文章を書くときに使用する P要素 、タイトルなどの見出しに使用する H1 ~ H6 までのヘディング要素、リンクを作成する A要素 、要素をひとかたまりの範囲として定義する DIV要素 、要素をひとつの範囲として定義する SPAN要素 、画像を貼る IMG 、他のページを埋め込む IFRAME要素 などが 代表的なHTML要素です。

HEAD要素 には、ブラウザに表示されない情報を記述します。
例えば、ページタイトル、 TITLE要素 ＜title＞ここにページタイトルを記述する＜/title＞ メタ情報 META要素 (ページの説明 meta name="description" content="HTMLを誰が書いたのか、どんなウェブサイトなのかとか、ここに記入した情報が検索サイトの説明欄などに表示されます。" や 作者名 meta name="author" content="文書の作者名" など)、 関連ファイルとの関係 （ファビコン link rel="icon" href="画像データURL" が、独自のファビコンへのリンク や CSS 別ページ CSSの基本 で詳しく解説します。）など を記述します。

これは、ここに紹介したすべての要素をまとめたHTMLです。


  <!DOCTYPE html>
  <html lang="ja">
  <head>
  <title>ここにページタイトルを記述する</title>
  
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="author" content="文書の作者名">
  <meta name="description" content="ページの説明">

  <link rel="icon" href="画像データURL">
  <link rel="stylesheet" href="css"/>
  
  <style>
  <-- ここにCSSを記述 -->
  </style>
  </head>
  
  <body>
  <p>you can create a paragraph by writing some text and P most elements are</p>
  <p class="special">we're giving the paragraph a class of special and it</p>
  <a href="#">href="url"</a>
  <h1>h1 - h6 element is the heading contents.</h1>
  <br/>
  <img src="画像データURL" alt="src=">
  
  <div id="">
  <span style=""></span>
  </div>
  </body>
  </html>

このHTMLと、 sample.html を比較し、HTMLがどのようにブラウザに表示されているかを理解できれば、HTMLの基本は完璧です。
