RUBY 要素

ベーステキストの上、下、隣に描画される小さな注釈をつける要素

ここでは、ルビを振るテキストの範囲全体を、 RUBY要素 で囲み、RP要素・RT要素を使ってルビを振る方法と、開始タグ内に属性data-ruby="ルビ"を記述してルビを振る方法を解説します。

RUBY RP: ルビのフォールバック用括弧要素 RT: ルビ文字列要素 


          <p>
              The
              <ruby>
            ruby
            <rp>(</rp><rt>HyperText Markup Language element</rt><rp>)</rp>
          </ruby> represents small annotations that are rendered above, below, or next to base text, usually used for showing the pronunciation of East Asian characters.
          </p>

ルビを振る文字全体をRUBYで囲み、ルビの前後の()を含むRP要素を記述、ルビをRT要素で囲みます。

この方法だと要素が多く複雑なので、CSSを用いてシンプルにした記述方法も併せて紹介します。


<p>
<ruby data-ruby="The HTML ruby element">
  ルビ
</ruby> はベーステキストの上、下、隣に
<ruby data-ruby="びょうが">
  描画
</ruby> される小さな
<br />
<ruby data-ruby="本文の語句や文章をとりあげてその意味を解説すること。また、その解説したもの。注。注解。">
  注釈
</ruby> で、よく東アジアの文字の
<ruby data-ruby="音、特に、言語の音声を発すること。 また、その音声の出し方。 はついん。">
  発音
</ruby> を表すのに使われます。
</p>
<p>
    他の種類の注釈にも使われることがありますが、この使い方はあまり一般的ではありません。
</p>

ルビを振りたい文字をRUBY要素で囲み、開始タグ内に属性data-ruby="ルビ"を記述。

ルビをCSS 擬似要素を用いて、表示したい位置にスタイリングします。

CSSでデザイン調整可能なルビ 


ruby {
    position: relative;
    font-size: 1.25rem;
    line-height: 150%;
}

[data-ruby]::before {
    content: attr(data-ruby);
    position: absolute;
    font-size: 50%;
    line-height: 100%;
}

以上が、レラティブポジションのRUBY要素に、アブソリュートポジションの擬似要素を生成し、ルビを振るCSSです。


縦書きレイアウト

テキストの行のレイアウトをプロパティ writing-mode で縦書きに指定し、プロパティ text-orientation で行内のテキストの向きを設定する。 


ruby {
  writing-mode: vertical-rl;
  text-orientation: mixed;
}
