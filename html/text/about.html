代表的なテキスト要素

このページでは、代表的なテキスト要素を解説します。


    <p>: 段落要素</p>

    <hgroup>
        <h1> is the heading contents</h1>
        <h2> is the heading contents</h2>
        <h3> is the heading contents</h3>
        <h4> is the heading contents</h4>
        <h5> is the heading contents</h5>
        <h6> is the heading contents</h6>
    </hgroup>

    <br/>
    <span>
        <b>: 注目付け要素</b>
        <i>: 興味深いテキスト要素</i>
        <u>: 非言語的注釈 (下線) 要素</u>
        <s> 要素は取り消し線を引いた文字列を表示します。</s>
    </span>
    <br/>
    <span>
    <q>: 行内引用要素</q>
    <samp class="fontmotion">: サンプル出力要素</samp>
    <small>: 附随コメント要素</small>
    <strong>: 強い重要性要素</strong>
    <sub>: 下付き文字要素</sub>
    <sup>: 上付き文字要素</sup>
  </span>

CSS
選択した要素の最初と最後に擬似要素 ::before (:before) ::after (:after) を生成。 


#pseudo-elements p::before {
    content: '<p>';
}

#pseudo-elements p::after {
    content: '</p>';
}
