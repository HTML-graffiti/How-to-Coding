A要素とCLASS・ID属性

このページでは、別のウェブページ、ファイル、メールアドレス、同一ページ内の場所へのハイパーリンクを作成するために使用する A: アンカー要素 についてと、要素を装飾するために特定の名前をつけるグローバル属性と呼ばれるHTML言語 CLASS と ID についてを解説します。

HTMLを見る

このHTMLのBODY要素は、ID名CLICKと名付けたP要素と、その中にある複数のCLASS・ID名がついたA要素で構成されています。


    <a href="/coding/html/href/howto.html" target="_blank" rel="noopener noreferrer">A</a>
    <p id="click">
      <a id="one" class="none" href="#two">click</a>
      <a id="two" class="randomcolor border" href="#three">click</a>
      <a id="three" class="randombg" href="#four">click</a>
      <a id="four" class="border" href="#five">click</a>
      <a id="five" class="randomcolor randombg radius none" href="#one">click</a>
    </p>

HTML内に、同じ属性名をつけること／一つの要素に複数の属性名をつけることができるCLASS属性に対して、同じID名を同じHTML内で使用することは適切ではありません。
加えて、ID属性の大きな特徴は、A要素のリンク先にID名を指定できることです。このページ内のA要素をクリックするとページ内のIDへ移動するようにA要素とID名がリンクします。

※ ID名 CLICK の前に記述したA要素は、target="_blank" rel="noopener noreferrer" と記述することで、リンク先を別タブで表示するように指定しています。

CSSを見る

ID名#CLICK内のA要素をフルディスプレイサイズで表示するCSS


      #click a {
        display:block;
        width:90%;
        height:90vh;
        margin:5vh auto;
      }

display width height margin

ID名#CLICK内のA要素内のテキストを中央に表示するCSS


      #click a {
        font-size:2rem;
        text-align:center;
        line-height:90vh;
      }

font-size text-align line-height color

CLASS名NONEの装飾 text-decoration を消し、CLASS名BORDERを線 border で囲い、CLASS名RADIUSを角が丸い点線で囲み、影をつける


      #click .none {
        text-decoration:none;
      }
      #click .border {
        border:solid;
      }
      #click .radius {
        border:dashed;
        border-radius: 2.5rem;
        box-shadow: 1rem 1rem 1rem #000;
      }

border border-radius box-shadow

スムーススクロール

ページ内リンク先へ動的に移動するスムーススクロールという仕様を実現するJavaScript


<script>
$(function(){
   // #で始まるアンカーをクリックした場合に処理
   $('a[href^=#]').click(function() {
      // スクロールの速度
      var speed = 1000; // ミリ秒
      // アンカーの値取得
      var href= $(this).attr("href");
      // 移動先を取得
      var target = $(href == "#" || href == "" ? 'html' : href);
      // 移動先を数値で取得
      var position = target.offset().top;
      // スムーススクロール
      $('body,html').animate({scrollTop:position}, speed, 'swing');
      return false;
   });
});
</script>
