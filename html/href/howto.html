A要素とCLASS・ID属性

このページでは、別のウェブページ、ファイル、メールアドレス、同一ページ内の場所へのハイパーリンクを作成するために使用する A: アンカー要素 についてと、要素を装飾するために特定の名前をつけるグローバル属性と呼ばれるHTML言語 CLASS と ID についてを解説します。

HTMLを見る

このHTMLのBODY要素は、ID名CLICKと名付けたP要素と、その中にある複数のCLASS・ID名がついたA要素で構成されています。


    <a href="/coding/html/href/howto.html" target="_blank" rel="noopener noreferrer">A</a>
    <p id="click">
      <a id="one" class="none" href="#two">click</a>
      <a id="two" class="randomcolor border" href="#three">click</a>
      <a id="three" class="randombg" href="#four">click</a>
      <a id="four" class="border" href="#five">click</a>
      <a id="five" class="randomcolor randombg radius none" href="#one">click</a>
    </p>

HTML内に、同じ属性名をつけること／一つの要素に複数の属性名をつけることができるCLASS属性に対して、同じID名を同じHTML内で使用することは適切ではありません。
加えて、ID属性の大きな特徴は、A要素をのリンク先として指定できることです。このページ内のA要素をクリックするとページ内のIDへ移動するようにA要素とID名がリンクします。

※ ID名 CLICK の前に記述したA要素は、target="_blank" rel="noopener noreferrer" と記述することで、リンク先を別タブで表示するように指定しています。

CSSを見る

CSSでスタイリングを行わない場合、A要素には下線が付き、文字色は青で表示されます。
また、そのA要素のリンク先にすでに訪問したことがある場合、文字・下線は紫色で表示されます。

はじめに、ID名#CLICK内のA要素をフルディスプレイサイズで表示するため、 display width height margin のプロパティ値を記述。
次に、 font-size を記述し、要素の真ん中に文字を表示するように text-align line-height のプロパティ値を記述。


      #click a {
        display:block;
        width:90%;
        height:90vh;
        margin:5vh auto;
        font-size:2rem;
        text-align:center;
        line-height:90vh;
      }

A要素の上をカーソル (マウスポインター) がホバー (通過)した時 :hover 、リンク先にすでに訪問したことがある場合 :visite 、A要素をクリックした際 :active に、要素の色が反転するようにCSSのfilterプロパティ値 invert() を記述しスタイリング


      #click a:hover,
      #click a:visited {
        filter: invert(1);
      }
      #click a:active {
        filter: invert(100%);
      }

最後に、CLASS名の違いを明確にするため、CLASS名NONEの装飾 text-decoration を消し、CLASS名BORDERを線 border で囲い、CLASS名RADIUSを角が丸い点線で囲み、影をつける border border-radius box-shadow CSSを記述


      #click .none {
        text-decoration:none;
      }
      #click .border {
        border:solid;
      }
      #click .radius {
        border:dashed;
        border-radius: 2.5rem;
        box-shadow: 1rem 1rem 1rem #000;
      }

以上がA要素とCLASS・ID属性の解説です。

        function getRumRgba(){
          var clr = 'rgba(';
          for(i=0; i < 3; i++){
            clr = clr + Math.floor( Math.random()*255) + ',';
          }
          clr = clr + Math.floor( Math.random()*10)/10 + ')';
          return clr;
        }

色コードをRGBで指定し、1から255までの数字をランダムで生成し、


          $(function(){
            jQuery('.randombg').css({'background':getRumRgba()});
          });
          
          $(function(){
            jQuery('.randomcolor').css({'color':getRumRgba()});
          })
