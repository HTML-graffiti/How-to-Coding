A要素とCLASS・ID属性

このページでは、別のウェブページやファイル、同一ページ内の要素へのハイパーリンクを作成する A: アンカー要素 と、要素に特定の名前をつけるグローバル属性と呼ばれるHTML言語 CLASS と ID についてを解説します。

HTMLを見る

このHTMLのBODY要素には、ID名CLICKと名付けたP要素と、その中にある複数のCLASS・ID名がついたA要素があります。


    <p id="click">
        <a id="one" class="shadow" href="#two">click</a>
        <a id="two" class="solid none radius" href="#three">click</a>
        <a id="three" class="double shadow" href="#four">click</a>
        <a id="four" class="dotted radius none" href="#five">click</a>
        <a id="five" class="radius none shadow" href="#one" .html ">click</a>
    </p>

同じHTML内に、同じCLASS名の要素を作ること／一つの要素に複数のCLASS名をつけることができるCLASS属性に対して、同じHTML内で同じID名をつけること／一つの要素に複数のID名をつけることは適切ではありません。
加えて、ID属性の大きな特徴は、A要素のリンク先にID名を指定できることです。

※ このページ内のA要素は、ID名がついたA要素とリンクしているため、A要素をクリックするとリンクしたA要素へページ内を移動します。

CSSを見る

※ CSSでは、ID名を「#ID名」CLASS名を「.CLASS名」 と記述します。

ID名#CLICK内のA要素をフルディスプレイサイズで表示し、要素の上下に空白を作るCSS


#click {
    display: block;
}
#click a {
    display: inline-block;
    width: 90%;
    height: 90vh;
    padding: 2.5vh 2.5%;
    margin: 2.5vh auto;
}

display width height padding margin

CSSプロパティ「ディスプレイ」については、DIVとSPAN要素 で詳しく解説します。

要素の大きさ・余白について、コンピューターとモバイルで表示内容を変える で詳しく解説します。


ID名#CLICK内のA要素のテキストを要素の中央に表示するCSS


#click a {
    font-size: 5vw;
    text-align: center;
    vertical-align: middle;
    line-height: 90vh;
}

プロパティ値
font-size
文字の大きさを指定するプロパティ
 text-align
水平方向の配置を設定するプロパティ
 vertical-align
垂直方向の配置を設定するプロパティ
 line-height
テキストの行間の高さを指定するプロパティ

font-size text-align vertical-align line-height


アンカー要素をスタイリングするCSS


a {
    text-decoration: underline;
}

a:link {
    color: blue;
}

a:visited {
    color: purple;
}

a:hover {
    background: yellow;
}

a:active {
    color: red;
}

a:focus {
    outline: solid;
}

アンカー要素には、 擬似クラス と呼ばれる選択された要素に対して特定の状態を指定するCSSがデフォルトで指定されています。
:link
まだ訪問していないA要素
 :visited
すでに訪問したリンクを含むA要素
 :hover
カーソル (マウスポインター) を要素の上でかざしたとき
 :active
アクティブ化されている要素（主ボタンを押し下げたとき）
 :focus
フォーカスを持っている要素（要素をクリックやタップをしたり、キーボードの Tab キーで選択したりしたとき）

:link :visited :hover :active :focus


要素に境界線をスタイリングするCSS 


#click .none {
    text-decoration: none;
}

#click .solid {
    border: solid;
}

#click .double {
    border: double;
}

#click .dotted {
    border: dotted;
}

#click .shadow {
    box-shadow: 1rem 1rem 1rem #000;
}

#click .radius {
    border-radius: 2.5rem;
}

border
要素の境界を設定するプロパティ
 border-radius
要素の境界の外側の角を丸めるプロパティ
 border-width
要素の境界の幅を設定するプロパティ
 border-collapse
セルが境界を共有するか分離するかを設定設定するプロパティ
 border-spacing
隣り合うセルの境界同士の間隔を定めるプロパティ

border border-radius border-width border-collapse border-spacing

テキストを装飾するCSSプロパティは text-decoration で詳しく解説します。
