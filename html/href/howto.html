A要素とCLASS・ID属性

このページでは、別のウェブページ、ファイル、メールアドレス、同一ページ内の要素へのハイパーリンクを作成する A: アンカー要素 と、要素に特定の名前をつけるグローバル属性と呼ばれるHTML言語 CLASS と ID についてを解説します。

HTMLを見る

このHTMLのBODY要素には、ID名CLICKと名付けたP要素と、その中にある複数のCLASS・ID名がついたA要素があります。


    <p id="click">
        <a id="one" class="shadow" href="#two">click</a>
        <a id="two" class="solid none radius" href="#three">click</a>
        <a id="three" class="double shadow" href="#four">click</a>
        <a id="four" class="dotted radius none" href="#five">click</a>
        <a id="five" class="radius none shadow" href="#one" .html ">click</a>
    </p>

同じHTML内に、同じCLASS名の要素を作ること／一つの要素に複数のCLASS名をつけることができるCLASS属性に対して、同じHTML内で同じID名の要素を作ること／一つの要素に複数のID名をつけることは適切ではありません。
加えて、ID属性の大きな特徴は、A要素のリンク先にID名を指定できることです。

※ このページ内のA要素は、ID名がついたA要素とリンクしているため、A要素をクリックするとリンクしたA要素へページ内を移動します。

CSSを見る

※ CSSでは、ID名を「#ID名」CLASS名を「.CLASS名」 と記述します。

ID名#CLICK内のA要素をフルディスプレイサイズで表示し、要素の上下に空白を作るCSS


#click a {
    display: inline-block;
    width: 90%;
    height: 90vh;
    padding: 2.5vh 2.5%;
    margin: 2.5vh auto;
}

display width height padding margin


ID名#CLICK内のA要素のテキストを要素の中央に表示するCSS


#click a {
    font-size: 5vw;
    text-align: center;
    vertical-align: middle;
    line-height: 90vh;
}

font-size text-align vertical-align line-height


ID名#CLICK内にあるA要素をCLASS名によって装飾する 


#click .none {
    text-decoration: none;
}

#click .solid {
    border: solid;
}

#click .double {
    border: double;
}

#click .dotted {
    border: dotted;
}

#click .shadow {
    box-shadow: 1rem 1rem 1rem #000;
}

#click .radius {
    border-radius: 2.5rem;
}

border border-radius box-shadow text-decoration
