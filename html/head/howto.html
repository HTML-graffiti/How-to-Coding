HEADって何？

HTML の文書の head 部分は、ページが読み込まれてもウェブブラウザーには表示されない部分です。

例えば、 タイトルバーやページのタブに表示される文書の題名を定義する TITLE: 文書題名要素、 HTMLの作成者・説明を記述、HTML文書の文字コードを指定する メタデータ: META要素 など、検索エンジンでページを上位にできる可能性がある情報（この行為に対する用語は Search Engine Optimization (検索エンジン最適化) または SEO といいます）や、CSSやサイトのアイコン(favicon)などの外部のリソースとの関係 LINK: 外部リソースへのリンク要素 を指定するなど、ブラウザーには表示されないけど重要な情報を記述し、HTMLについてよい基礎を与えます。 


    <head>
    <title>ここにページタイトルを記述する</title>
    
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="author" content="文書の作者名">
    <meta name="description" content="HTMLを誰が書いたのか、どんなウェブサイトなのかとか、ここに記入した情報が検索サイトの説明欄などに表示されます。">
    
    <link rel="icon" href="画像データURL">
    <link rel="stylesheet" href="css"/>
    
    <style>
    <-- ここにCSSを記述 -->
    </style>
    </head>
CSSを見る

※ CSSでは、ID名を「#ID名」CLASS名を「.CLASS名」 と記述します。

ID名#CLICK内のA要素をフルディスプレイサイズで表示し、要素の上下に空白を作るCSS


      #click a {
        display:inline-block;
        width:90%;
        height:90vh;
        margin:5vh auto;
      }

display width height margin

ID名#CLICK内のA要素のテキストをフォントサイズ2remで要素の中央に表示するCSS


      #click a {
        font-size:2rem;
        text-align:center;
        vertical-align:middle;
        line-height:90vh;
      }

font-size text-align vertical-align line-height

ID名#CLICK内にあるCLASS名.NONEの装飾 text-decoration を消し、ID名#CLICK内にあるCLASS名.BORDERを線 border で囲い、ID名#CLICK内にあるCLASS名.RADIUSを角が丸い点線で囲み、影をつけるCSS


      #click .none {
        text-decoration:none;
      }
      #click .border {
        border:solid;
      }
      #click .radius {
        border:dashed;
        border-radius: 2.5rem;
        box-shadow: 1rem 1rem 1rem #000;
      }

border border-radius box-shadow
