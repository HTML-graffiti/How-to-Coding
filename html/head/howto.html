HEADって何？

HTML の文書の head 部分は、ページが読み込まれてもウェブブラウザーには表示されない部分です。

例えば、 タイトルバーやページのタブに表示される文書の題名を定義する TITLE: 文書題名要素、 HTMLの作成者・説明を記述、HTML文書の文字コードを指定する メタデータ: META要素 など、検索エンジンでページを上位にできる可能性がある情報（この行為に対する用語は Search Engine Optimization (検索エンジン最適化) または SEO といいます）や、CSSやサイトのアイコン(favicon)などの外部のリソースとの関係を指定する LINK: 外部リソースへのリンク要素 などを記述します。 


    <p id="click">
      <a id="one" class="none" href="#two">click</a>
      <a id="two" class="border" href="#three">click</a>
      <a id="three" class="radius" href="#four">click</a>
      <a id="four" class="border" href="#five">click</a>
      <a id="five" class="radius" href="#one">click</a>
    </p>
  <a class="demo" href="/coding/html/href/hello.html" target="_blank" rel="noopener noreferrer">HTMLのみ（CSSをリンクしない場合）の表示を見る</a>

同じHTML内に、同じCLASS名の要素を作ること／一つの要素に複数のCLASS名をつけることができるCLASS属性に対して、同じHTML内で同じID名の要素を作ること／一つの要素に複数のID名をつけることは適切ではありません。
加えて、ID属性の大きな特徴は、A要素のリンク先にID名を指定できることです。

※ 左ページ内のA要素は、ID名がついたA要素とリンクしているため、A要素をクリックするとリンクしたA要素へページ内を移動します。

CSSを見る

※ CSSでは、ID名を「#ID名」CLASS名を「.CLASS名」 と記述します。

ID名#CLICK内のA要素をフルディスプレイサイズで表示し、要素の上下に空白を作るCSS


      #click a {
        display:inline-block;
        width:90%;
        height:90vh;
        margin:5vh auto;
      }

display width height margin

ID名#CLICK内のA要素のテキストをフォントサイズ2remで要素の中央に表示するCSS


      #click a {
        font-size:2rem;
        text-align:center;
        vertical-align:middle;
        line-height:90vh;
      }

font-size text-align vertical-align line-height

ID名#CLICK内にあるCLASS名.NONEの装飾 text-decoration を消し、ID名#CLICK内にあるCLASS名.BORDERを線 border で囲い、ID名#CLICK内にあるCLASS名.RADIUSを角が丸い点線で囲み、影をつけるCSS


      #click .none {
        text-decoration:none;
      }
      #click .border {
        border:solid;
      }
      #click .radius {
        border:dashed;
        border-radius: 2.5rem;
        box-shadow: 1rem 1rem 1rem #000;
      }

border border-radius box-shadow
