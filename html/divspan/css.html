DIVとSPAN要素はとても似ています。
どちらもHTML内のコンテンツを区分するための要素で、大きな違いは DISPLAYプロパティの値 です。

DISPLAY は、要素の表示方法を指示するCSSプロパティです。

display:block; は、要素の上下にスペースを生成し、要素を一つの段落として区分する ブロックレベル要素 として表示します。
DIV要素 のデフォルトは display:block; です。

display:inline; は、一つの文章の中にある要素を「」のように区切るため、ブロックレベル要素と異なり、要素は新しい行から始まらず、上下にスペースを作りません。
SPAN要素 のデフォルトは display:inline; です。

※block では指定できる、要素の大きさ 幅（width）と高さ（height）、文字の位置（text-align・vertical-align）などの指定が、inlineでは出来ません。


display:inline-block; は、新しい行から始まらず、前後にスペースを作らないinlineと同じ表示方法で、ブロック要素と同じようにCSSプロパティを加えることができるディプレイプロパティ値です。
例えば、IMG要素のデフォルトは display:inline-block; です。

※inline では指定できない、要素の大きさ 幅（width）と高さ（height）、文字の位置（text-align・vertical-align）などの指定が、inline-block では可能になります。

style.css

これは、DIVとSPAN要素を羅列したHTMLです。


  span {
    font-size:1rem;
    border:solid 1px;
  }
  .both {
    color:#fff;
    background:#000;
  }
  .pc {
    color:green;
    background:yellow;
  }
  .mobile {
    color:red;
    background:blue;
  }
  .mobile {display:none;}


  @media screen and (max-width: 500px){
    .mobile {display:block;}
    .pc {display:none;}

    span {
      padding:1rem;
      margin:1rem;
    }
    .both {
      font-size:2rem;
    }
    .mobile {
      font-size:3rem;
    }
  }

CLASS名「.both」要素の背景色は黒・文字色は白
CLASS名「.pc」要素の背景色は緑・文字色は黄
CLASS名「.mobile」要素の背景色は赤・文字色は青
にスタイリングしています。

※ 加えて、CLASS名「.mobile」要素のディスプレイレベルを display:none; に指定しているため、ウインドウサイズ 500px 以上のブラウザでは、要素が表示されません。


@media screen and (max-width: 500px){ より下は、ウインドウサイズ 500px 以下のブラウザに対応するCSSです。

ウインドウサイズ 500px 以下のブラウザでは、CLASS名が .mobile の要素はSPAN要素でもブロックレベル要素として表示されます。

※ 加えて、CLASS名「.pc」要素のディスプレイレベルを display:none; に指定しているため、ウインドウサイズ 500px 以下のブラウザでは、要素が表示されません。

そのほか、CLASS名によってフォントサイズが変化するようにCSSを記述しています。
