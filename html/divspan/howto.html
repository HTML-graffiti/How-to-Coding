DIVとSPAN要素

DIVとSPAN要素はとても似ています。
どちらもHTML内のコンテンツを区分するための要素で、大きな違いは DISPLAYプロパティの値 です。

DISPLAY は、要素の表示方法を指示するCSSプロパティです。

display:block; は、要素の上下にスペースを生成し、要素を一つの段落として区分する ブロックレベル要素 として表示します。
DIV要素 のデフォルトは display:block; です。

display:inline; は、一つの文章の中にある要素を「」のように区切るため、ブロックレベル要素と異なり、要素は新しい行から始まらず、上下にスペースを作りません。
SPAN要素 のデフォルトは display:inline; です。

※ ブロック要素は、要素の大きさ 幅（width）と 高さ（height）や、文字の位置（text-align・vertical-align）などを指定できます。
インライン要素は、「」のように区切るだけなので、それらを指定することができません。


display:inline-block; は、新しい行から始まらず、前後にスペースを作らないinlineと同じ表示方法で、ブロック要素と同じようにCSSプロパティを加えることができるディプレイプロパティ値です。
例えば、IMG要素のデフォルトは display:inline-block; です。

※inline では指定できない、要素の大きさ 幅（width）と高さ（height）、文字の位置（text-align・vertical-align）などの指定が、inline-block では可能になります。

上記のHTMLのように、DIV要素の中にSPAN要素を記述する（ブロック要素の中にインライン要素を含む）ことはありますが、 逆に、SPAN要素の中にDIV要素を記述する（インライン要素の中にブロック要素を含む）ことは一般的ではありません。

DIVとSPAN要素に「.both」「.pc」「.mobile」とCLASS名をつけ、ディスプレイレベル（プロパティ値）によってスタイリングが異なるレスポンシブCSSを記述します。

コードを見る


<meta name="viewport" content="width=device-width, initial-scale=1.0">

初めに、HEAD要素 に上記の一行を記述し、HTMLをレスポンシブ仕様のページにします。


display.css


span {
    font-size: 50%;
    border: solid 1px;
}

.pc {
    color: green;
    background: yellow;
}

.mobile {
    color: red;
    background: blue;
}

.mobile {
    display: none;
}

@media screen and (max-width: 900px) {
    .mobile {
        display: block;
    }
    .pc {
        display: none;
    }
}

CLASS名「.both」要素の背景色・文字色はJavaScriptによりランダム生成
CLASS名「.pc」要素の背景色は緑・文字色は黄
CLASS名「.mobile」要素の背景色は赤・文字色は青
にスタイリングしています。

※ 加えて、CLASS名「.mobile」要素のディスプレイレベルを display:none; に指定しているため、ウインドウサイズ 900px 以上のブラウザでは、CLASS名「.mobile」要素は表示されません。


@media screen and (max-width: 900px) { } ウインドウサイズ 900px 以下のブラウザに対応するCSSです。

ウインドウサイズ 900px 以下のブラウザでは、CLASS名「.mobile」要素は、SPAN要素でもブロックレベル要素として表示されます。

※ 加えて、CLASS名「.pc」要素のディスプレイレベルを display:none; に指定しているため、ウインドウサイズ 900px 以下のブラウザでは、CLASS名「.pc」要素は表示されません。

以上がDIVとSPAN要素の違い、また、表示方法を指定するCSSプロパティ・ディスプレイ値の解説です。

要素の色をスタイリングするCSSは、COLOR ページで詳しく解説します。
