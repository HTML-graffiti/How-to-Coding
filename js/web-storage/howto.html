Web Storage には、2 種類の仕組みがあります

sessionStorage
ページのセッション中のみデータを保存します。（データがサーバーに転送されることはありません。）

localStorage
有効期限なしでデータを保存します。（ブラウザーを閉じたり再び開いたりしてもデータは持続します。） 

// Web Storage にデータを保存する
sessionStorage.setItem('key', 'value');
localStorage.setItem('key', 'value');


// Web Storage に保存したデータを取得する
let data = sessionStorage.getItem('key');
let data = localStorage.getItem('key');

// Web Storage に保存したデータを削除する
sessionStorage.removeItem('key')
localStorage.removeItem('key');

// Web Storage に保存したすべてのデータを削除する
localStorage.clear();
